我们在后续的拓扑量子场论中还会说明继续怎么描述一个整体的拓扑的性质.

虽然人们认识到时空几何的过程是崎岖而又充满物理思辨的，但是我们初识这门历经百年的学科，完全可以借助人们在这百年里发展的更加本质更加简洁的途径，狭义相对论的物理本质，就是描述时空的几何
性质.


我们先来说说在物理上我们如何描述几何.要描述时空这个概念，我们在局域上实际要描述的是所谓的“距离”，给定两个物体，如何从我给它的坐标中得到距离呢？

以欧式几何为例，我们实际上要把握的就是$ds^2=dx^2+dy^2+dz^2$.
这个式子直接给定了在无穷小尺度上距离如何计算，在不考虑整体的非局域的拓扑性质时，这便是给定了一个流形的几何.

我们先来说说什么是流形，局域上每个部分都能等价于实数域的直积$\h R^{\otimes n}$的拓扑空间，在其上不同部分到$\h R^
n$上的映射，在这些部分相交的地方，你可以构建出一个$\h R^n$到$\h R^n$的映射，我们要求这个映射是一个无穷可微的函数. 其中，拓扑空间的意思为，我给定其中的任意一块区域，任意一个子集，
我可以判断其是不是开集，几何意义上说就是能否判断连续.如果读者以后学习比较系统的数学，就会发现连续性，甚至实数的定义戴德金分割等等，都用开集的语言来叙述.
而局域上等价于实数域的直积，也就是每个点都隶属于某个子开集的区域，这个子开集区域的特点是可以一一映射到$\h R^n$的某个开集区域上，那么这个点可以用某个坐标
$(x_1,x_2,\cdots,x_n)$来描述.这里需要说明的点是，不是每个开集都要存在这样的映射，而是对于每个点，都存在某个开集包含它，并且有这样的映射.这样的子开集被我们称作图
(chart)，而任何能够覆盖整个空间的开集（不管是不是图）的集合被我们称作一个开覆盖，任何由图构成的开覆盖被称作图表(atlas)，我们对于微分流形（即流形）的定义就是，这个空间起码可以给定
一个图表，图表的不同图间是兼容的，即两个图的相交部分，各自对某个点给定的两套坐标间可以一一映射起来，这个一一映射实际上给出了n个n元函数，这些函数都是$\h C^\infty$，即无穷可微的（可以一直求导数）.

所以，当我们谈论坐标时，我们实际上在谈论局域上这些图到$\h R^n$上的映射，我们可以把所有相容的图全都收录到大图表里面，比如，球坐标和笛卡尔坐标来描述欧式三维空间是相容的，其中这个大图表
里面各种子图表（包括它自身）都被我们认为在描述同一个流形.或许你们已经发现，虽然图是局域的，但当我考虑图表这个开覆盖的时候，已经在涉及一些非局域的整体的性质，实际上，一个拓扑空间
的开覆盖的信息会给出许多很重要的拓扑性质（这里不需要到流形程度的结构，只需要开覆盖，开集，而不需要图和图表），比如，如果空间的所有开覆盖中都能找出有限的开覆盖子集，那么这个拓扑空间
是紧致的，比如球面，而紧致与否其实是很重要的拓扑性质.

现在，我们可以说明$(x,y,z)$是什么了，它是我们取的欧式空间的图给定的坐标，由于这个空间性质足够的好，我们可以完整的用$(x,y,z)$来描述整个空间的点.现在我们接着来说明，什么是操作$d$.

值得庆幸的是，在物理里面，起码我接触到的各种内容从来不需要严格区分什么是算子什么是算符等等，只需要分清楚物理量和物理量在特别基矢量上的表示这套表示论，就可以足够严谨的避开所有错误，
所以我们不需要长篇大论的说明，当然$d$的含义还是足够丰富，它会在多个地方不时出现，而且每次仿佛都多了新的含义，所以，我们最好首先说明什么是$dx$的含义，然后再说明另一个本质上相同
可以统一$dx$的运算$d$的含义，最后我将说明这个算符如何和电磁学的规范冗余联系起来，这部分的物理结构实质和量子规范场论的BRST量子化是相通的，虽然涉及的冗余不完全一样.

首先，我们需要问，在这个流形上怎么找到我们熟知的矢量（不是线性矢量那个矢量，虽然这个矢量是一个矢量），那个表征方向的“矢量”.我们借助在古典微积分中就已经熟知的方向导数的概念，
一个矢量的方向和长度可以用一个函数的变化来表现.所以我们首先构造标量场，或者标量函数，一个映射$f:\h M_n\rarr \h C \tx{or} \h R$，通过图给定的坐标，这可以被描述为一个n元函数，我们要求
这个函数无穷可微.所有这些无穷可微的映射可以构成一个集合，我们现在可以定义表征方向的切矢量为这样一些映射$v_p:f\rarr\h C \tx{or} \h R,f\in\{map\  \phi: \h M_n\rarr \h C \tx{or} 
\h R|\phi \tx{光滑}\},p\in \h M_n $.

所以切矢量是每个点上将“将流形上每个点映射到一个数的映射”映射到一个数的映射，这个操作也被我们称作对n元函数相对于某点的切矢量求导，这样的好处是求导的定义是个一般的映射，不依赖于
撕出很多腥风血雨的无穷小语言.现在我们可以说出$dx$是什么了，他代表的是一类新的矢量，是一个将切矢量映射到一个数的线性映射，我们称之为对偶矢量.现在让我们来将某抽象的切矢量
定义为沿x方向求导$\partialderivative{}{x}$，我们认为这个映射由古典微积分中的n元函数对x求偏导给出，或者恰好吻合（毕竟按我们的定义所有这样的映射都是切矢量，所以一定
可以找到这个特定的切矢量），这样我们就完美的避开了无穷小，但是发现了满足我们想要的“求导”规则的求导.
现在我们再规定我们标记为$dx$的特殊对偶矢量，是将$\partialderivative{}{x}$恰好映射到1的对偶矢量.
